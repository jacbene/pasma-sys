# Pasma-sys 🏫

Application de gestion des cotisations APEE et suivi scolaire.

## 📋 Fonctionnalités

- **Tableau de bord** : Vue d'ensemble des statistiques
- **Cotisations APEE** : Gestion des paiements et reçus
- **Enseignants** : Annuaire des professeurs titulaires
- **Suivi scolaire** : Notes, absences, devoirs, emploi du temps
- **Facturation** : Gestion des frais (cantine, transport, inscription)
- **Messagerie** : Communication avec l'administration et les enseignants

## 🚀 Installation

```bash
# Cloner le projet
git clone https://github.com/jacbene/pasma-sys.git
cd pasma-sys

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
